About

Origami SMTP is a fake, or what some may say dummy or false, Simple Mail Transport Protocol server. It is provided free of charge to anyone who wants to use it. It is even free to distribute to your friends and co-workers. Also, it is written in Java to be operating system independent.

Features

HTML Message

Receive email messages that are encoded with HTML tags.

TLS Support

Receive emails over a secure connection with STARTTLS.

OS Independent

No matter your operating system; be Windows, Macintosh, or Linux; it will run on it.
